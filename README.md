# CarPulse

Закрытый MVP сервиса мониторинга автомобильных площадок. Пользователь создаёт радары вручную в личном кабинете, видит свежие совпадения в общей ленте и может подключить Telegram для мгновенных уведомлений.

## Что уже есть

- адаптивный личный кабинет;
- форма создания радара без ссылок на готовый поиск;
- лента найденных автомобилей и избранное;
- экран подключения Telegram по одноразовому коду;
- D1-схема для радаров, подключений Telegram и избранного;
- единый интерфейс адаптеров Auto24, SS.lv, Nettiauto и mobile.de.

Площадочные адаптеры пока намеренно оставлены неподключёнными: интерфейс работает на демонстрационных данных, а реальный мониторинг подключается по одному источнику после отдельной проверки его условий и технической устойчивости.

## Запуск

```bash
npm install
npm run dev
```

Проверка production-сборки:

```bash
npm run build
npm test
```

После изменения `db/schema.ts`:

```bash
npm run db:generate
```
